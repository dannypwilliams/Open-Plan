# Phase 4 Art Report

The final office uses 47 Blender-generated assets across reception/elevator, twelve desks, meeting room, manager area, coffee/break zone, water corner, utility/copier area, windows, walls, columns, signage, plants, and clutter. Shared URP materials, warm directional shadows, cool/cyan practicals, and the dark surrounding void preserve miniature readability without a greybox layer.

Performance remained 119.88 fps average at 1080p in the release probe. All fifteen required gameplay views were captured from the actual camera.
